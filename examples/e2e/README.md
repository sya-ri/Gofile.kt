# e2e: End-to-End Test

Exhaustively verify whether the API works. It is for development and may not be useful as an example.

## Environments

| Key            | Value                                                              |
|----------------|--------------------------------------------------------------------|
| `GOFILE_TOKEN` | Gofile.io API token in the [profile](https://gofile.io/myProfile). |

## Scripts

### Build

```shell
./gradlew build
```

### Run application

```shell
./gradlew run
```
