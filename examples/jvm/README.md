# jvm: File uploader

A simple file upload CLI tool.

## Environments

| Key            | Value                                                              |
|----------------|--------------------------------------------------------------------|
| `GOFILE_TOKEN` | Gofile.io API token in the [profile](https://gofile.io/myProfile). |

## Scripts

### Build

```shell
./gradlew build
```

### Run application

```shell
./gradlew run
```
